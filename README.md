## Sobre o Projeto

Este assistente virtual permite que você faça perguntas por voz sobre cotações de moedas, câmbio e indicadores financeiros. Ele utiliza:

- Whisper (OpenAI) -> Transcrição de áudio para texto
- ChatGPT (OpenAI) -> Geração de respostas inteligentes
- gTTS (Google) -> Conversão de texto para áudio

## Funcionalidades

- Perguntas por voz sobre dolar, euro, bitcoin e outras moedas
- Respostas em audio com cotacoes e explicacoes
- Prompt personalizado para atuar como especialista em cambio
- Modular e facil de adaptar para APIs reais de cotacao
